# WhatToDov1

Projektmitglied (Solo):
Bilal Hamaydi

Diese Website soll eine ToDo-Liste mit einem Kalender verbinden.

Funktionen:
- Tasks erstellen (done)
- Kalender aufrufen
- Tasks kategorisieren (done)
- Kategorien erstellen
- Kategorien löschen 
- Tasks löschen (done)
- Tasks abhaken (done)
- Farben für Tasks (done)
- Dark-Mode (done)

Optional:
- Benachrichtigungen


*Mindestanforderungen:*

Web-App nach Render deployed, öffentlich ansprechbar. (done)

Es werden Entitäten in einer Datenbank gespeichert, die über
die Web-Oberfläche von Nutzern erstellt werden können. (done)

Es gibt sowohl Frontend- als auch Backend-Tests und diese
werden automatisiert von GitHub ausgeführt (CI/CD)

Credentials gehen nicht als Klartext nach GitHub. (done)

App setzt grob die ursprünglich präsentierte Projekt-Idee um
und nutzt vorgegebenen Tech-Stack (Spring, Vue, Postgres).
